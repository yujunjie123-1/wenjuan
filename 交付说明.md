# 本地问卷演示填报器交付说明

## 项目定位

本项目是合规版本地问卷演示填报工具，只用于授权问卷、本地样例问卷或测试环境。

不包含以下能力：

- IP 代理池
- 设备指纹伪装
- 隐藏浏览器自动化特征
- 高速并发提交
- 绕过第三方平台限制

## 已实现功能

- 本地网页入口：`http://localhost:8080`
- Excel 上传与前 10 行预览
- Excel 列与问卷题目字段映射
- 支持填空题、单选题、多选题、下拉题
- Playwright Java 调用本机 Chrome 可见浏览器
- 顺序逐行填写
- 默认不点击提交按钮
- 可选点击提交按钮
- 成功/失败日志
- 失败原因、行号、原始行数据和截图记录
- 本地样例问卷：`http://localhost:8080/sample-form.html`

## 技术栈

- Kotlin
- Ktor
- Apache POI
- Playwright Java
- 原生 HTML/CSS/JavaScript 前端
- Gradle Wrapper

## 启动方式

进入项目目录后执行：

```powershell
.\gradlew.bat run
```

然后打开：

```text
http://localhost:8080
```

## 样例 Excel 列

```text
姓名    性别    年级    满意度    建议
张三    男      大二    满意      无
李四    女      大三    一般      希望改进
```

## 样例映射

| Excel 列 | 问卷题目 | 题型 |
| --- | --- | --- |
| 姓名 | 请输入你的姓名 | 填空题 |
| 性别 | 你的性别 | 单选题 |
| 年级 | 你的年级 | 下拉题 |
| 满意度 | 你是否满意 | 单选题 |
| 建议 | 你的建议 | 填空题 |

## 主要文件

- `build.gradle.kts`：项目依赖和构建配置
- `settings.gradle.kts`：Gradle 项目配置
- `src/main/kotlin/com/localform`：后端源码
- `src/main/resources/static`：本地网页前端和样例问卷
- `src/main/resources/logback.xml`：日志配置
- `README.md`：基础运行说明
